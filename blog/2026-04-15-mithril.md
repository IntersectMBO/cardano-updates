## High level overview

This week, the Mithril team completed several SNARK-related items: they created the recursive SNARK circuit new module in the STM library, worked on support for test mode for the recursive circuit, refactored the bytes codec for forward and backward compatibility, synchronized the non-recursive circuit prototype updates to the STM library, prepared the testing plan for the IVC circuit, enhanced the witness preparation for the non-recursive prover, and added golden tests for the recursive SNARK circuit prototype.

They completed partial block range support, security parameter offset signing, and the WASM client implementation for Cardano blocks and transactions. They also continued progressing on the explorer update and the documentation of the Cardano blocks and transactions signature on the website. Additionally, they continued work on the upgrade to Cardano `10.7` and completed the DMQ node updates to `0.4.1.0` and `0.4.2.0`.

Finally, the team completed cross-compiled Windows builds from Linux using Nix, fixed some node build failures in Hydra CI, and investigated the SNARK aggregator certification stoppage on the dev network.
